A **frontend-only weather application** with user authentication. Users can register, log in, and view weather information for Bangladeshi cities using the **OpenWeather API**. The app features a modern, glassmorphism-style UI with responsive design for mobile and desktop devices.

---

## üîπ Features

- **User Authentication**
  - Register a new account
  - Login with email & password
  - Logout functionality
- **Weather Dashboard**
  - Displays weather for major cities in Bangladesh by default
  - Search for any Bangladeshi city
  - Weather details include:
    - Temperature
    - Humidity
    - Wind speed
    - Weather description
- **Responsive Design**
  - Optimized for mobile, tablet, and desktop
  - Glassmorphism UI style
  - Neon buttons and hover animations
- **Frontend Only**
  - Uses **LocalStorage** for user authentication
  - Uses **OpenWeather API** for fetching weather data
  - No backend required
